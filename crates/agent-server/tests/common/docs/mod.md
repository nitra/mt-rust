## Публічний API

**WsStream** — Створює та керує потоком WebSocket-з'єднання.
**connect** — Ініціює підключення до WS-клієнта, обмінюється `ClientHello` та `ServerHello`, повертає активний стрім. `device_id` використовується для ідентифікації клієнта в сценаріях з мультипідключеннями.
**next_json** — Зчитує наступний текстовий кадр зі стріму як десеріалізований об'єкт `T` з тайм-аутом 10 секунд.
