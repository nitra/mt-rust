## Огляд

Overview
Файл відповідає за управління життєвим циклом вузла через мутації `mt invalidate` та `mt kill` для архівування та видалення даних.

Behavior
child_nodes: Отримує директорію і повертає список імен вузлів, які містять `task.md`.
invalidate: Архівує version chain вузла та каскадно архівує всіх нащадків.
kill: Архівує весь вузол з нащадками у директорію історії і видаляє директорію.
