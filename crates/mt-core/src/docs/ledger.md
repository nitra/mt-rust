## Огляд

Огляд
Файл призначений для збору та агрегування метрик часу та токенів з усіх вузлів воркспейсу. Файл фіксує сировину для звітності, агрегуючи дані `wall_sec`, `tokens_in`, `tokens_out` та `cost_usd` з усіх записів `run_NNN.md` для забезпечення GUI-аналітики на Фазі 4.

## Поведінка

Поведінка
LedgerEntry: Агрегує метрики одного вузла.
CostLedger: Зберігає агреговані метрики по всьому воркспейсу.
build_cost_ledger: Будує ledger сканом усього дерева воркспейсу та `run_NNN.md` кожного вузла.
