## Огляд

Файл підтримує Markdown-документи із секцією «Суть» і fragment-блоками рушія, щоб читати, оновлювати й зберігати стислий зміст документа.

Експортовані константи-рядки: `DRAFT_MARKER="<!-- essence:draft -->"` — позначає секцію «Суть» як чернетку, коли документ ще потребує уточнення змісту.

## Поведінка

- `DRAFT_MARKER` — позначає секцію «Суть» як чернетку значенням `<!-- essence:draft -->`.

- `parseDoc` — відокремлює frontmatter від тіла Markdown-документа й повертає тіло без змін, якщо коректний frontmatter присутній; інакше вважає весь текст тілом.

- `serializeDoc` — збирає Markdown-документ із frontmatter і тіла у стабільному для власних документів форматі.

- `extractEssence` — знаходить секцію «Суть», повертає її текст і ознаку чернетки; якщо секції немає, повертає порожній результат.

- `insertEssenceDraft` — додає чернетку секції «Суть» після головного заголовка та вступного blockquote, щоб документ отримав місце для стислого змісту.

- `parseSourceLine` — перетворює source-рядок із шляхом і двома CRC-значеннями на структурований запис джерела; невалідний рядок відхиляє помилкою.

- `formatSourceLine` — перетворює структурований запис джерела назад у source-рядок для маркерів fragment-блоку.

- `extractFragment` — знаходить fragment-блок рушія в Markdown-тексті, повертає назву шару, список джерел і внутрішній вміст; якщо маркерів немає або блок неповний, повертає порожній результат.

- `replaceFragment` — замінює наявний fragment-блок новим списком джерел і вмістом, залишаючи решту файлу недоторканою; якщо потрібних маркерів немає, завершується помилкою.

## Публічний API

- DRAFT_MARKER — позначає автоматично вставлену чернетку секції «Суть».
- parseDoc — відокремлює frontmatter документа від основного вмісту, зберігаючи тіло без змін.
- serializeDoc — збирає frontmatter і тіло назад у документ зі стабільним повторним читанням для власних docs рушія.
- extractEssence — дістає зі змісту документа секцію «Суть».
- insertEssenceDraft — додає чернеткову секцію «Суть» після головного заголовка та вступної цитати.
- parseSourceLine — розкладає рядок джерела fragment-маркера на шлях і діапазон рядків.
- formatSourceLine — перетворює опис джерела fragment-блоку назад у компактний рядок.
- extractFragment — знаходить у тексті службовий fragment-блок рушія.
- replaceFragment — оновлює fragment-блок: переписує список джерел у маркері та замінює вміст між межами.
